## 审查类别

1. **色情内容 (sexual_content)**
   - 露骨的性表现、裸体、淫秽描写
   - 违反日本刑法第175条（传播淫秽物品）的内容
   - 暗示儿童性剥削的内容（严格判定）

2. **暴力・残忍 (violence)**
   - 过度暴力描写、流血、身体损伤
   - 酷刑、虐待描写
   - 鼓励自伤行为

3. **仇恨・歧视 (hate_discrimination)**
   - 基于种族、民族、国籍的歧视
   - 基于性取向、性别认同的歧视
   - 残障歧视
   - 使用仇恨符号

4. **违法内容 (illegal)**
   - 促进违禁药物使用・销售
   - 赞美反社会势力
   - 鼓励诈骗・犯罪行为

5. **不当广告 (misleading_ad)**
   - 违反景品表示法（优良误认、有利误认）
   - 违反药机法（标注未获批的功效）
   - 夸大广告

6. **个人隐私 (privacy)**
   - 暴露可识别个人身份的信息
   - 未经授权使用肖像

## 输出格式

请务必按以下JSON格式输出：

```json
{
  "verdict": "PASS|FAIL|REVIEW",
  "confidence": 0.0-1.0,
  "categories": [
    {
      "category": "类别名称",
      "severity": "none|low|medium|high|critical",
      "detail": "具体说明"
    }
  ],
  "summary": "审查结果摘要（中文）"
}
```

- **PASS**：无问题
- **REVIEW**：需要人工复核
- **FAIL**：违反策略，不可发布

请对每个类别都进行判定。没有问题的类别请标记 severity: "none"。
